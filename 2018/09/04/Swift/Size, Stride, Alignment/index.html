Swift에서 메모리를 다룰때 3가지 유형의 프로퍼티 타입을 고려해야합니다.

MemoryLayout

MemoryLayout을 통해서 메모리 유형을 확인 할 수 있습니다.
해당 유형에서 구조체의 크기를 찾으려면 generic 매개 변수와 함께 size 속성을 사용하면 됩니다.

1. Size

  • 구조체 안의 전체 프로퍼티의 타입의 size를 계산 합니다.

Int32

인스턴스가있는 경우 size(ofValue:) 함수를 사용하면 됩니다.

Stride

  • 구조체 안의 전체 프로퍼티중 가장 큰 프로퍼티 사이즈의 2배의 사이즈를 갖는다.(Padding)

배열과 같은 단일 버퍼 내부에서 여러 인스턴스를 처리 할 때 Stride는 중요 해집니다.

Puppy는 9바이트의 크기를 가지고 있는데 연속적인 배열안에 있다면 Memorys는 어떻게 생겼을까요?

결과적으로 그렇지 않습니다.

Stride는 두 요소 사이의 거리를 결정하는데 이 값은 크기가 더 크거나 같습니다.

16???
프로퍼티 중 가장 큰 사이즈가 Int이기 떄문에 Int의 사이즈인 8의 사이즈에 2배를 가지게 됩니다.

실질적인 메모리의 모습은 이렇습니다.

즉, 첫 번째 요소에 대한 바이트 포인터가 있고 두 번째 요소로 이동하려는 경우 Stride는 포인터를 이동시키는 데 필요한 바이트 거리 수입니다.

Alignment

구조체 유형의 Aignment는 모든 속성 중 최대 크기의 프로퍼티를 타나냅니다. Int와 Bool 사이에서 Int는 8의 더 큰 정렬 값을 가지므로 구조체는 이를 사용합니다.
